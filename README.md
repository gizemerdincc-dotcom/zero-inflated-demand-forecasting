# zero-inflated-demand-forecasting
Neural forecasting framework for sparse and zero-inflated demand prediction

This project presents a simplified forecasting framework for sparse and zero-inflated demand data.

The approach combines:
- temporal features
- demand occurrence modeling
- neural forecasting concepts
- explainable prediction logic

## Motivation

Short-horizon demand forecasting often suffers from:
- sparse observations
- intermittent demand
- excessive zero values

Traditional forecasting models struggle under these conditions.

## Objective

The goal is to build a lightweight forecasting framework capable of:
- handling zero inflation
- improving demand estimation
- supporting operational decision-making

## Planned Components

- Data preprocessing
- Feature engineering
- Baseline comparison
- Forecast evaluation
- Visualization
